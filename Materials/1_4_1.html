Exact Equations: The Quest for Exactness

Exact equations are a special type of first-order ODE that can be solved directly by finding a potential function.

Key Concepts:

  • Definition: A first-order ODE of the form M(x, y) dx + N(x, y) dy = 0 is exact if ∂M/∂y = ∂N/∂x.
  • Potential Function: A function ϕ(x, y) such that ∂ϕ/∂x = M and ∂ϕ/∂y = N.
  • Solution: The solution is given by ϕ(x, y) = C, where C is a constant.
  • Integrating Factors: If not exact, an integrating factor μ(x, y) may make it exact.

Example:

Consider the equation (2x + 3y) dx + (3x + 2y) dy = 0.

  • Check for exactness:
    M(x, y) = 2x + 3y, so ∂M/∂y = 3
    N(x, y) = 3x + 2y, so ∂N/∂x = 3
    Since ∂M/∂y = ∂N/∂x, it is exact.
  • Find the potential function:
    Integrate M with respect to x: 
    ϕ(x, y) = ∫ (2x + 3y) dx = x² + 3xy + h(y)
    Differentiate ϕ with respect to y: 
    ∂ϕ/∂y = 3x + h'(y)
    Set equal to N: 
    3x + h'(y) = 3x + 2y → h'(y) = 2y
    Integrate h'(y): 
    h(y) = ∫ 2y dy = y²
    Thus: 
    ϕ(x, y) = x² + 3xy + y²

The solution is: x² + 3xy + y² = C.

Linear Equations: The General Solution

Linear first-order ODEs have a standard form and a structured method to find the solution.

Key Concepts:

  • Standard Form: dy/dx + P(x)y = Q(x)
  • Integrating Factor: μ(x) = e^(∫P(x) dx)
  • General Solution: y(x) = (1/μ(x)) ∫ μ(x)Q(x) dx

Example:

Consider the equation dy/dx + 2y = e-x.

  • Identify:
    P(x) = 2, Q(x) = e-x
  • Find the integrating factor: 
    μ(x) = e^(∫2 dx) = e^(2x)
  • Find the general solution: 
    
y(x) = (1/e^(2x)) ∫ e^(2x) * e^(-x) dx
     = (1/e^(2x)) ∫ e^(x) dx
     = (1/e^(2x)) (e^(x) + C)
     = e^(-x) + Ce^(-2x)

Bernoulli’s Equation: Transformation to Linearity

Bernoulli’s equation is nonlinear but can be transformed into a linear equation.

Key Concepts:

  • General Form: dy/dx + P(x)y = Q(x)yⁿ
  • Substitution: v = y^(1-n)
  • Transformed Equation: dv/dx + (1-n)P(x)v = (1-n)Q(x)

Example:

Consider the equation dy/dx + (1/x)y = xy³.

  • Identify:
    P(x) = 1/x, Q(x) = x, n = 3
  • Substitution:
    Let v = y^(1-3) = y^(-2)
    Then: 
    dv/dx = -2y^(-3) dy/dx
  • Substituting into the original equation: 
    
-1/2 dv/dx + (1/x)y = xy³
-1/2 dv/dx + (1/x)(y) = x(y³)
    Rearranging and using v = y^(-2) gives: 
    dv/dx - (2/x)v = -2x
  • Solve the linear equation: 
    
Integrating factor: μ(x) = e^(∫-2/x dx) = x^(-2)

v(x) = (1/x^(-2)) ∫ x^(-2)(-2x) dx
     = x² ∫ -2x^(-1) dx
     = x²(-2ln|x| + C)
  • Substituting back: 
    y(x) = ± 1 / sqrt(x²(-2ln|x| + C))

Applications: Real-World Modeling

First-order ODEs model many real-world phenomena:

  • Newton’s Law of Cooling:
    Models temperature change.
    dT/dt = k(T - Tₛ)
  • Growth and Decay Models:
    Models population growth, radioactive decay.
    dN/dt = kN
  • R-L Circuits:
    Models current in electrical circuits.
    L dI/dt + RI = V(t)
