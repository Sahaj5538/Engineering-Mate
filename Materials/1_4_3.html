Double Integrals

Evaluation:

Double integrals are used to find the volume under a surface (z = f(x, y)) over a region (R) in the (xy)-plane. The double integral is denoted as:

∬R f(x,y) dA

where (dA) can be (dx dy) or (dy dx).

To evaluate a double integral, we perform iterated integration. For example, if (R) is defined by (a ≤ x ≤ b) and (g₁(x) ≤ y ≤ g₂(x)), then:

∬R f(x,y) dA = ∫ab ∫g₁(x)g₂(x) f(x,y) dy dx

Example:

Evaluate ∬R (x + y) dA, where (R) is the region bounded by (x = 0), (x = 2), (y = 0), and (y = 1).

∬R (x+y) dA = ∫02 ∫01 (x+y) dy dx

First, integrate with respect to (y):

∫01 (x+y) dy = [xy + (1/2)y²]01 = x + 1/2

Now, integrate with respect to (x):

∫02 (x + 1/2) dx = [(1/2)x² + (1/2)x]02 = 2 + 1 = 3

Thus, ∬R (x + y) dA = 3.

Changing Order of Integration:

Sometimes, it's easier to evaluate a double integral by changing the order of integration. This involves rewriting the limits of integration.

Example:

Evaluate ∫01 ∫x1 ey² dy dx.

Since it's difficult to integrate ey² with respect to (y), we change the order of integration. The region (R) is defined by (0 ≤ x ≤ 1) and (x ≤ y ≤ 1). Rewriting this, we have (0 ≤ y ≤ 1) and (0 ≤ x ≤ y).

∫01 ∫x1 ey² dy dx = ∫01 ∫0y ey² dx dy

Now, integrate with respect to (x):

∫0y ey² dx = y ey²

Next, integrate with respect to (y):

∫01 y ey² dy

Let u = y² ⇒ du = 2y dy ⇒ (1/2) du = y dy.

Thus, the integral becomes (1/2) ∫01 eu du = (1/2)(e1 - e0) = (1/2)(e - 1).

Thus, ∫01 ∫x1 ey² dy dx = (1/2)(e - 1).

Variable Changes (Cartesian to Polar):

When integrating over regions with circular symmetry, it's often easier to use polar coordinates:

  • x = r cos(θ)
  • y = r sin(θ)

and dA = r dr dθ.

Example:

Evaluate ∬R e-(x² + y²) dA, where (R) is the disk x² + y² ≤ 4.

In polar coordinates, x² + y² = r², and R is 0 ≤ r ≤ 2, 0 ≤ θ ≤ 2π.

Thus, ∬R e-(x² + y²) dA = ∫02π ∫02 e-r² r dr dθ.

First, integrate with respect to (r):

∫02 e-r² r dr = -(1/2)[e-r²]02 = (1/2)(1 - e-4)

Now, integrate with respect to (θ):

∫02π (1/2)(1 - e-4) dθ = π(1 - e-4)

Thus, ∬R e-(x² + y²) dA = π(1 - e-4).

Triple Integrals

Evaluation:

Triple integrals are used to find the volume of a region (V) in 3D space or to integrate a function f(x, y, z) over (V). The triple integral is denoted as:

∭V f(x,y,z) dV

where dV can be (dx dy dz), (dy dx dz), etc.

Example:

Evaluate ∭V x dV, where (V) is the region bounded by (x = 0), (x = 1), (y = 0), (y = 2), (z = 0), (z = 3).

∭V x dV = ∫01 ∫02 ∫03 x dz dy dx

First, integrate with respect to (z):

∫03 x dz = 3x

Now, integrate with respect to (y):

∫02 3x dy = 6x

Finally, integrate with respect to (x):

∫01 6x dx = 3

Thus, ∭V x dV = 3.

Changes of Variables:

Cylindrical Coordinates:

  • x = r cos(θ)
  • y = r sin(θ)
  • z = z

and dV = r dz dr dθ.

Spherical Coordinates:

  • x = ρ sin(ϕ) cos(θ)
  • y = ρ sin(ϕ) sin(θ)
  • z = ρ cos(ϕ)

and dV = ρ² sin(ϕ) dρ dϕ dθ.

Example (Spherical Coordinates):

Evaluate ∭V e-(x² + y² + z²)3/2 dV, where V is x² + y² + z² ≤ 1.

In spherical coordinates:

∭V e-(ρ²)3/2 dV = ∫02π ∫0π ∫01 e-ρ³ ρ² sin(ϕ) dρ dϕ dθ

First, integrate with respect to (ρ):

∫01 e-ρ³ ρ² dρ = (1/3)(1 - e-1)

Now, integrate with respect to (ϕ):

∫0π (1/3)(1 - e-1) sin(ϕ) dϕ = (2/3)(1 - e-1)

Finally, integrate with respect to (θ):

∫02π (2/3)(1 - e-1) dθ = (4π/3)(1 - e-1)

Thus, ∭V e-(x² + y² + z²)3/2 dV = (4π/3)(1 - e-1).

Applications

  • Finding Areas: A = ∬R dA
  • Finding Volumes: V = ∭V dV
  • Mass and Center of Mass: m = ∭V ρ(x, y, z) dV, where ρ is the density function.
  • Moments of Inertia: I = ∭V r² ρ(x, y, z) dV, where r is the distance from the axis of rotation.
