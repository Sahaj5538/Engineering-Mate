Mathematical Logic and Predicates

Mathematical Logic

Statements and Notations

A statement is a declarative sentence that is either true or false, but not both. Statements are often denoted by letters such as p, q, r.

Connectives

Logical connectives are used to combine statements:

  • AND ( ∧ ) - Conjunction
  • OR ( ∨ ) - Disjunction
  • NOT ( ¬ ) - Negation
  • IMPLICATION ( → ) - If...Then
  • BICONDITIONAL ( ↔ ) - If and only if

Well-Formed Formulas

A well-formed formula (WFF) is a string of symbols that is grammatically (syntactically) correct according to the rules of the logical language.

Truth Tables

Truth tables are used to determine the truth value of a compound statement for every possible truth value of its component statements.

Tautology

A tautology is a compound statement that is always true, regardless of the truth values of the individual statements.

Equivalence and Implication

  • Logical Equivalence ( ≡ ) means two statements always have the same truth value.
  • Implication ( → ) states that if one statement is true, then another is also true.

Normal Forms

Statements can be expressed in specific standard forms:

  • Conjunctive Normal Form (CNF) - a conjunction of disjunctions
  • Disjunctive Normal Form (DNF) - a disjunction of conjunctions

Predicates

Predicative Logic

Predicative logic extends propositional logic by dealing with predicates and quantifiers. A predicate is a statement that contains variables.

Example: P(x): x is greater than 5.

Free and Bound Variables

  • Free variable: A variable not bound by a quantifier.
  • Bound variable: A variable that is quantified (using ∀ or ∃).

Rules of Inference

Rules of inference are the logical rules that dictate the valid steps from premises to conclusions:

  • Modus Ponens: If p → q and p are true, then q is true.
  • Modus Tollens: If p → q and ¬q are true, then ¬p is true.
  • Hypothetical Syllogism: If p → q and q → r are true, then p → r is true.

Consistency

A set of statements is consistent if there is no contradiction among them. If all statements can be true simultaneously, they are consistent.

Proof by Contradiction

In proof by contradiction, we assume the negation of what we want to prove. If this leads to a contradiction, then the original statement must be true.

Example:
To prove: √2 is irrational
Assume: √2 is rational (can be expressed as p/q)
Derive contradiction → Hence √2 is irrational.
