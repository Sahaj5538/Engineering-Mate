Combinational Logic

Binary Adder/Subtractor

A Binary Adder/Subtractor is a combinational circuit that performs addition and subtraction on binary numbers.

Types of Adders:

  • Half Adder: Adds two single binary digits and produces a sum and carry output.
  • Full Adder: Adds three binary digits (including carry input) and produces a sum and carry output.

Subtractor:

  • Half Subtractor: Subtracts two bits and produces a difference and borrow.
  • Full Subtractor: Subtracts three bits (including borrow input) and produces a difference and borrow output.

Adder/Subtractor Circuit:

A combined circuit can perform both addition and subtraction based on a control input.

  • If control = 0: Perform addition.
  • If control = 1: Perform subtraction (using 2's complement method).

Multiplexers & Demultiplexers

Multiplexers (MUX):

A multiplexer is a combinational circuit that selects one of many input signals and forwards it to a single output line based on select lines.

  • Example: A 4-to-1 MUX selects one of 4 input lines using 2 select lines.
  • Useful for data routing and resource sharing in digital circuits.

  • Example: A 1-to-4 DEMUX distributes an input signal to one of 4 outputs.
  • Used for data distribution and signal demultiplexing.

Decoders & Encoders

Decoders:

A decoder is a combinational circuit that converts binary information from n input lines to a maximum of 2n unique output lines.

  • Example: A 2-to-4 decoder has 2 inputs and 4 outputs.
  • Used in memory address decoding and data routing.

Encoders:

An encoder is a combinational circuit that converts 2n input lines into an n-bit binary code.

  • Example: A 4-to-2 encoder takes 4 inputs and outputs 2-bit binary code.
  • Used in keyboard inputs, priority systems, etc.

Magnitude Comparators

A magnitude comparator is a combinational circuit that compares two binary numbers and determines their relative magnitudes.

Functions:

  • Checks if one number is greater than, less than, or equal to another number.

Outputs:

  • A > B
  • A = B
  • A < B

Magnitude comparators are used in sorting networks, arithmetic operations, and decision-making circuits.
