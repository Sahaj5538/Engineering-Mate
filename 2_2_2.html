UNIT II: Classes, Inheritance, and Polymorphism

Classes and Objects

Classes

A class is a blueprint for creating objects. It defines properties (attributes) and methods (functions).

Objects

Objects are instances of classes. They contain real data and can perform actions defined by their methods.

Creating Objects

ClassName obj = new ClassName();

Methods

Methods define the behavior of objects and can perform operations or return values.

Constructors

Constructors are special methods used to initialize new objects.

Constructor Overloading

Having multiple constructors with different parameters.

Garbage Collector

Java automatically destroys unused objects to free up memory using a process called Garbage Collection.

Class Variables and Methods (Static Keyword)

When a variable or method is declared as static, it belongs to the class rather than instances.

this Keyword

The this keyword refers to the current object within an instance method or constructor.

Arrays

Arrays are data structures that store multiple values of the same type.

int[] numbers = new int[5];

Command Line Arguments

Command-line arguments are passed to the main method and can be accessed via the args[] array.

Nested Classes

Classes defined within another class are called nested classes. They help in logically grouping classes.

Strings in Java

String

Immutable sequence of characters.

StringBuffer

Mutable sequence of characters; used when frequent modifications are required.

StringTokenizer

Used to split strings into tokens based on delimiters.

import java.util.StringTokenizer;

StringTokenizer st = new StringTokenizer("Hello World", " ");
while (st.hasMoreTokens()) {
    System.out.println(st.nextToken());
}

Inheritance and Polymorphism

Types of Inheritance

  • Single Inheritance
  • Multilevel Inheritance
  • Hierarchical Inheritance
  • Multiple Inheritance (via interfaces)

Deriving Classes using extends Keyword

The extends keyword is used to create a subclass from a superclass.

class Animal {
    void sound() {
        System.out.println("Animal makes a sound");
    }
}

class Dog extends Animal {
    void bark() {
        System.out.println("Dog barks");
    }
}

Polymorphism

Polymorphism allows objects to be treated as instances of their parent class rather than their actual class.

Method Overloading

Same method name with different parameter lists within the same class.

Method Overriding

Subclass provides a specific implementation of a method declared in its superclass.

class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}

class Dog extends Animal {
    @Override
    void sound() {
        System.out.println("Dog barks");
    }
}

final Keyword

The final keyword is used to:

  • Declare constants
  • Prevent method overriding
  • Prevent inheritance

Abstract Classes

An abstract class cannot be instantiated and can contain abstract methods (methods without a body).

abstract class Animal {
    abstract void sound();
}

class Dog extends Animal {
    void sound() {
        System.out.println("Dog barks");
    }
}
