UNIT I: Object Oriented Thinking and Java Basics

Object Oriented Thinking

Introduction

Object-oriented thinking focuses on modeling real-world entities as objects that combine data and behavior. It helps in designing modular, reusable, and scalable software systems.

Need for Object-Oriented Programming

  • Promotes modularity and code reuse.
  • Helps manage complexity in large software systems.
  • Encourages better data protection (encapsulation).
  • Enhances flexibility and maintainability of code.

Principles of Object-Oriented Languages

  • Encapsulation: Wrapping data and methods together.
  • Abstraction: Hiding complex details and exposing only essential parts.
  • Inheritance: Acquiring properties and behaviors from a parent class.
  • Polymorphism: Ability to take multiple forms (method overloading/overriding).

Applications of OOP

  • Software development (desktop, web, mobile apps).
  • Game development.
  • Simulations and modeling systems.
  • Artificial Intelligence and Machine Learning.

Java Overview

History of Java

Java was developed by James Gosling and his team at Sun Microsystems in 1995. Initially called "Oak," it was later renamed "Java" after the Java coffee.

Java Virtual Machine (JVM)

The JVM is an abstract computing machine that enables a computer to run Java programs by converting bytecode into machine code at runtime.

Java Features

  • Simple and Easy to Learn
  • Platform Independent (Write Once, Run Anywhere)
  • Object-Oriented
  • Robust and Secure
  • Multithreaded
  • High Performance (Just-In-Time Compiler)
  • Distributed

Program Structure

A basic Java program includes:

  • Class Definition
  • Methods (main method is the entry point)
  • Statements and Expressions
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Installation of JDK

  • Download Java Development Kit (JDK) from the official Oracle website or OpenJDK.
  • Install and set up environment variables (like JAVA_HOME and add to PATH).
  • Verify installation using java -version and javac -version in the terminal or command prompt.

Java Basics

Variables

Variables are containers for storing data values in Java.

Primitive Data Types

  • byte, short, int, long (integers)
  • float, double (floating-point numbers)
  • char (single character)
  • boolean (true/false)

Identifiers and Naming Conventions

  • Names can contain letters, digits, underscores, and dollar signs.
  • Cannot start with a digit.
  • Should be meaningful and use camelCase for variables and methods (e.g., studentName).

Keywords

Reserved words in Java like class, public, static, void, etc.

Literals

Fixed values assigned to variables, such as numbers, characters, or strings (e.g., 10, 'A', "Hello").

Operators

  • Binary Operators: +, -, *, /, %, &, |, ^, etc.
  • Unary Operators: +, -, ++, --, !
  • Ternary Operator: condition ? trueResult : falseResult

Expressions

Combination of variables, literals, operators that evaluate to a single value.

Primitive Type Conversion and Casting

Implicit Casting (Widening): Smaller to larger data type (e.g., int to double).

Explicit Casting (Narrowing): Larger to smaller data type (e.g., double to int), using syntax like (int).

Flow of Control

  • Branching: if, if-else, nested if-else, switch.
  • Conditional: Ternary operator.
  • Loops: for, while, do-while.
