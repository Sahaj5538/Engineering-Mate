Discrete and Continuous Probability Distributions

A probability distribution describes how the probabilities are distributed over the values of the random variable. Probability distributions are classified as:

  • Discrete Probability Distributions: Associated with discrete random variables (finite or countable set of outcomes).
  • Continuous Probability Distributions: Associated with continuous random variables (infinite outcomes, uncountable, measurable).

1. Binomial Distribution (Discrete)

Definition:

The binomial distribution models the number of successes in a fixed number of independent Bernoulli trials, each with the same probability of success.

Probability Mass Function (PMF):

P(X = x) = C(n,x) * p^x * (1-p)^(n-x)
where:

  • n = number of trials
  • p = probability of success
  • x = number of successes
  • C(n,x) = n! / (x!(n-x)!)

Properties:

  • Discrete random variable
  • Fixed number of trials (n)
  • Only two outcomes: success or failure
  • Constant probability p
  • Independent trials

Statistical Parameters:

  • Mean (μ) = n * p
  • Variance (σ²) = n * p * (1-p)
  • Standard Deviation (σ) = √(n * p * (1-p))

Applications:

  • Quality control (number of defective products)
  • Medical trials (success rate of treatments)
  • Survey analysis (number of positive responses)

2. Poisson Distribution (Discrete)

Definition:

The Poisson distribution models the number of occurrences of an event in a fixed interval of time or space when events occur independently and at a constant rate.

Probability Mass Function (PMF):

P(X = x) = e^(-λ) * λ^x / x!
where:

  • λ = average rate of occurrence
  • x = number of occurrences

Properties:

  • Discrete random variable
  • Events occur one at a time
  • Events are independent
  • Average rate (λ) is constant

Statistical Parameters:

  • Mean (μ) = λ
  • Variance (σ²) = λ
  • Standard Deviation (σ) = √λ

Applications:

  • Number of calls at a call center per hour
  • Number of decay events per second from radioactive material
  • Number of arrivals at a service station

3. Normal Distribution (Continuous)

Definition:

The normal distribution is a continuous probability distribution characterized by a symmetric, bell-shaped curve centered at the mean μ.

Probability Density Function (PDF):

f(x) = (1 / (σ√2π)) * e^(-(x-μ)^2 / (2σ²))
where:

  • μ = mean
  • σ² = variance
  • σ = standard deviation

Properties:

  • Continuous random variable
  • Symmetric about the mean μ
  • Mean = Median = Mode
  • Bell-shaped curve
  • Total area under curve = 1
  • Empirical Rule:
    • 68% of data within 1σ
    • 95% of data within 2σ
    • 99.7% of data within 3σ

Statistical Parameters:

  • Mean (μ) = Center of distribution
  • Variance (σ²) = Spread of the distribution
  • Standard Deviation (σ) = √Variance

Applications:

  • Height and weight distribution
  • Standardized test scores
  • Errors in measurements

4. Exponential Distribution (Continuous)

Definition:

The exponential distribution models the time between events in a Poisson process, where events occur continuously and independently at a constant average rate.

Probability Density Function (PDF):

f(x; λ) = λ * e^(-λx) for x ≥ 0
where:

  • λ = rate parameter

Properties:

  • Continuous random variable
  • Memoryless property: P(X > s + t | X > s) = P(X > t)
  • Mean = 1/λ
  • Variance = 1/λ²

Applications:

  • Modeling the lifespan of electronic components
  • Time between customer arrivals
  • Time until radioactive decay events

Evaluation of Statistical Parameters

Binomial Distribution:

  • Mean: μ = n * p
  • Variance: σ² = n * p * (1-p)
  • Standard Deviation: σ = √(n * p * (1-p))

Poisson Distribution:

  • Mean: μ = λ
  • Variance: σ² = λ
  • Standard Deviation: σ = √λ

Normal Distribution:

  • Mean: μ = expected value
  • Variance: σ² = expected squared deviation from mean
  • Standard Deviation: σ = √Variance

Note: In practice, knowing when to use which distribution is crucial. Discrete distributions are used when outcomes are countable, while continuous distributions apply when outcomes form a continuum (like time, weight, distance).
