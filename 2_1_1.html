Binary Systems & Boolean Algebra

Binary Numbers & Base Conversions

Binary numbers are numbers expressed in base-2 numeral system, using only two symbols: 0 and 1. Computers internally use binary to represent and manipulate all data.

Base Conversions:

  • Decimal to Binary: Divide the number by 2 repeatedly and record the remainders.
  • Binary to Decimal: Multiply each bit by 2 raised to its positional power and sum them up.
  • Binary to Hexadecimal: Group bits in 4s from right to left and convert each group to its hexadecimal equivalent.
  • Hexadecimal to Binary: Replace each hexadecimal digit with its 4-bit binary equivalent.

Signed Numbers & Complements

Signed numbers in binary allow for representation of both positive and negative integers.

  • Sign-Magnitude Representation: The most significant bit (MSB) represents the sign (0 for positive, 1 for negative), and the remaining bits represent the magnitude.
  • 1's Complement: Invert all the bits of a number to find its 1's complement.
  • 2's Complement: Take the 1's complement and add 1 to it. This is the most common method for representing signed numbers in computers.

Using 2's complement simplifies arithmetic operations like addition and subtraction in digital systems.

Binary Codes & Storage

Binary codes are systems of representing text, instructions, or other data using binary digits.

  • BCD (Binary Coded Decimal): Each decimal digit is represented by its 4-bit binary equivalent.
  • Gray Code: A binary numeral system where two successive values differ in only one bit. Used to prevent errors in digital circuits.
  • ASCII (American Standard Code for Information Interchange): A 7-bit character code used for text representation in computers.

Storage: Binary storage refers to how data is stored in memory devices using binary numbers (0s and 1s), such as in RAM, hard drives, or SSDs.

Boolean Algebra & Logic Gates

Boolean Algebra is a branch of algebra that deals with true or false values, usually represented as 1 and 0. It is fundamental to the design of digital circuits and computer systems.

Basic Boolean Operations:

  • AND (·): Output is true if both inputs are true.
  • OR (+): Output is true if at least one input is true.
  • NOT (¬ or '): Output is the inverse of the input.

Important Boolean Laws:

  • Identity Law
  • Null Law
  • Idempotent Law
  • Inverse Law
  • Distributive Law
  • Associative Law
  • Commutative Law
  • DeMorgan’s Theorems

Logic Gates:

Logic gates are physical devices implementing Boolean functions. Basic logic gates include:

  • AND Gate: Outputs 1 only if both inputs are 1.
  • OR Gate: Outputs 1 if at least one input is 1.
  • NOT Gate: Outputs the complement of the input.
  • NAND Gate: Output is the complement of AND.
  • NOR Gate: Output is the complement of OR.
  • XOR Gate: Outputs 1 if inputs are different.
  • XNOR Gate: Outputs 1 if inputs are the same.
