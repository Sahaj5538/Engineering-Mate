Set Theory, Functions, and Algebraic Structures

Set Theory

Properties of Binary Relations

  • Reflexive: Every element is related to itself.
  • Symmetric: If a is related to b, then b is related to a.
  • Transitive: If a is related to b and b is related to c, then a is related to c.
  • Antisymmetric: If a is related to b and b is related to a, then a = b.

Equivalence Relation

A relation that is reflexive, symmetric, and transitive.

Compatibility Relation

A relation that is reflexive and symmetric but not necessarily transitive.

Partial Ordering Relations

A relation that is reflexive, antisymmetric, and transitive.

Hasse Diagram

Graphical representation of a finite partially ordered set, showing the ordering relation without drawing redundant edges.

Functions

Inverse Function

If f: A → B is a bijection, then the inverse function f-1: B → A exists and satisfies f(f-1(b)) = b for all b in B.

Composite of Functions

If f: A → B and g: B → C, then the composition g∘f: A → C is defined by (g∘f)(a) = g(f(a)).

Recursive Functions

Functions that are defined using themselves within their definition.

Example:
Factorial(n) = 
    1 if n = 0
    n × Factorial(n-1) if n > 0

Lattice and its Properties

A lattice is a partially ordered set (poset) where every two elements have a unique supremum (join) and infimum (meet).

Pigeonhole Principle

If n items are put into m containers and n > m, then at least one container must contain more than one item.

Applications:

  • Proving existence results.
  • Finding duplicates in data sets.
  • Combinatorics and number theory problems.

Algebraic Structures

Algebraic Systems

A set equipped with one or more operations satisfying certain axioms.

Examples:

  • Groups
  • Rings
  • Fields
  • Vector Spaces

Semi Groups and Monoids

  • Semigroup: A set with an associative binary operation.
  • Monoid: A semigroup with an identity element.

Groups

A set G with a binary operation (·) satisfying:

  • Closure
  • Associativity
  • Identity element exists
  • Inverse element exists

Subgroups

A subset H of a group G that itself forms a group under the same operation.

Homomorphism

A function between two algebraic structures that preserves the operations.

Isomorphism

A bijective homomorphism. If there is an isomorphism between two structures, they are structurally identical.
