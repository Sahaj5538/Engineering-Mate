Functions: The Building Blocks of Structured Programming

Functions are self-contained blocks of code that perform a specific task. They are a cornerstone of structured programming, promoting code reusability, modularity, and maintainability.

Key Concepts:

Definition:

A function is a named block of code that performs a specific task.

Purpose:

  • Reusability: Write once, use multiple times.
  • Modularity: Break down complex tasks into smaller, manageable units.
  • Abstraction: Hide implementation details from the user.

Components:

  • Function Signature: Includes the function name, return type, and parameters.
  • Parameters: Input values passed to the function.
  • Return Type: The type of value the function returns.
  • Function Body: The code that performs the task.

Example:

Let's define a function to calculate the area of a rectangle:

FUNCTION CalculateArea(length, width)
  area = length * width
  RETURN area
ENDFUNCTION

// Usage
rectangleArea = CalculateArea(5, 10)
DISPLAY rectangleArea  // Output: 50

In this example:

  • CalculateArea is the function name.
  • length and width are the parameters.
  • The function returns the calculated area.

String Handling: Working with Text Data

Strings are sequences of characters, and string handling involves various operations to manipulate and process text data.

Common String Operations:

  • strlen(): Returns the length of a string.
  • strcat(): Concatenates (joins) two strings.
  • strcpy(): Copies one string to another.
  • strcmp(): Compares two strings.
  • substring(): Extracts a portion of a string.

Examples:

strlen():

string = "Hello, World!"
length = strlen(string)
DISPLAY length  // Output: 13

strcat():

string1 = "Hello, "
string2 = "World!"
result = strcat(string1, string2)
DISPLAY result  // Output: "Hello, World!"

strcpy():

string1 = "Hello"
string2 = strcpy(string1)
DISPLAY string2  // Output: "Hello"

strcmp():

string1 = "Hello"
string2 = "Hello"
result = strcmp(string1, string2)
IF result = 0 THEN
  DISPLAY "Strings are equal"
ELSE
  DISPLAY "Strings are not equal"
ENDIF
// Output: "Strings are equal"

substring():

string = "Hello, World!"
sub = substring(string, 0, 5)  // Start at index 0, length 5
DISPLAY sub  // Output: "Hello"

Putting It All Together

Functions and string handling are often used together to create powerful and flexible programs. For example, you might create a function that takes a string as input, performs some operations on it, and returns the modified string.
