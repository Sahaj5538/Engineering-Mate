Branching: Making Decisions

Branching allows your program to make decisions based on certain conditions. It's like choosing a path at a fork in the road.

1. If-Else Statements:

The if-else statement executes a block of code if a condition is true, and optionally another block if the condition is false.

Syntax:

IF condition THEN
  // Code to execute if condition is true
ELSE
  // Code to execute if condition is false
ENDIF

Example:

IF age >= 18 THEN
  DISPLAY "You are an adult"
ELSE
  DISPLAY "You are a minor"
ENDIF

2. Switch-Case Statements:

The switch-case statement is used to select one of several code blocks based on the value of a variable.

Syntax:

SWITCH variable DO
  CASE value1:
    // Code to execute if variable = value1
    BREAK
  CASE value2:
    // Code to execute if variable = value2
    BREAK
  DEFAULT:
    // Code to execute if no case matches
ENDSWITCH

Example:

SWITCH day DO
  CASE 1:
    DISPLAY "Monday"
    BREAK
  CASE 2:
    DISPLAY "Tuesday"
    BREAK
  DEFAULT:
    DISPLAY "Unknown day"
ENDSWITCH

Loops: Repeating Actions

Loops allow you to repeat a block of code multiple times. They are essential for automating repetitive tasks.

1. For Loops:

Syntax:

FOR i = start TO end DO
  // Code to execute
ENDFOR

Example:

FOR i = 1 TO 10 DO
  DISPLAY "Iteration: " + i
ENDFOR

2. While Loops:

Syntax:

WHILE condition DO
  // Code to execute
ENDWHILE

Example:

i = 1
WHILE i <= 10 DO
  DISPLAY "Iteration: " + i
  i = i + 1
ENDWHILE

3. Do-While Loops:

Syntax:

DO
  // Code to execute
WHILE condition

Example:

i = 1
DO
  DISPLAY "Iteration: " + i
  i = i + 1
WHILE i <= 10

Arrays: Storing Data

Arrays are used to store multiple data items of the same type in a contiguous block of memory.

  • Definition: A collection of elements of the same data type.
  • Accessing Elements: Elements are accessed using an index (starting from 0).

Example:

// Declare an array of 5 integers
numbers = [10, 20, 30, 40, 50]

// Access the first element
DISPLAY numbers[0]  // Output: 10

// Modify an element
numbers[2] = 35
DISPLAY numbers[2]  // Output: 35

Searching Algorithms: Finding Elements

1. Linear Search:

The linear search algorithm checks each element in the array one by one until the target element is found or the end of the array is reached.

Algorithm:

  • Start at the first element of the array.
  • Compare each element with the target element.
  • If a match is found, return the index of the element.
  • If the end of the array is reached without finding the target element, return -1.

Example:

FUNCTION linearSearch(array, target)
  FOR i = 0 TO array.length - 1 DO
    IF array[i] = target THEN
      RETURN i
    ENDIF
  ENDFOR
  RETURN -1
ENDFUNCTION

2. Binary Search:

The binary search algorithm is used to find a specific element in a sorted array by repeatedly dividing the search interval in half.

Algorithm:

  • Start with the entire array as the search interval.
  • Find the middle element of the interval.
  • If the middle element is the target, return its index.
  • If the target is less than the middle, search the left half.
  • If the target is greater, search the right half.
  • Repeat until the target is found or the interval is empty.

Example:

FUNCTION binarySearch(array, target)
  left = 0
  right = array.length - 1
  
  WHILE left <= right DO
    mid = (left + right) / 2
    
    IF array[mid] = target THEN
      RETURN mid
    ENDIF
    
    IF target < array[mid] THEN
      right = mid - 1
    ELSE
      left = mid + 1
    ENDIF
  ENDWHILE
  
  RETURN -1
ENDFUNCTION

Putting It All Together

These concepts are often used together to solve complex problems. For example, you might use a loop to iterate through an array and an if-else statement to make decisions based on the values in the array.
