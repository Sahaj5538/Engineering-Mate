UNIT I: Basic Structure of Computers and Register Transfer Language & Microoperations

Basic Structure of Computers

Computer Types

  • Supercomputers: Extremely powerful computers used for complex scientific calculations.
  • Mainframe Computers: Large systems used by enterprises for bulk data processing.
  • Minicomputers: Mid-sized systems used in business and scientific applications.
  • Microcomputers: Personal computers like desktops and laptops.

Functional Unit

  • Input Unit: Accepts coded information.
  • Output Unit: Communicates the processed results.
  • Memory Unit: Stores data and programs.
  • Arithmetic Logic Unit (ALU): Performs arithmetic and logic operations.
  • Control Unit: Directs the manner and sequence of operations.

Data Representation

  • Binary Numbers
  • Octal Numbers
  • Hexadecimal Numbers
  • Character Representation (ASCII, Unicode)

Fixed Point Representation

Represents numbers with a fixed number of digits before and after the decimal point. Used for integers and simple fractions.

Floating-Point Representation

Represents real numbers using a formula: number = mantissa × baseexponent. Follows IEEE 754 standard.

Error Detection Codes

  • Parity Bits: Single-bit error detection.
  • Checksum: Sum of data used for error detection.
  • CRC (Cyclic Redundancy Check): Detects accidental changes to raw data.
  • Hamming Code: Detects and corrects single-bit errors.

Register Transfer Language and Microoperations

Register Transfer Language (RTL)

Specifies the operations performed on data stored in registers and the control information required.

Register Transfer

Involves moving data from one register to another. Example notation: R2 ← R1 (contents of R1 are transferred to R2).

Bus and Memory Transfers

  • Bus: A shared communication path for transferring data between components.
  • Memory Transfer: Transfers data between registers and memory units.

Microoperations and Its Types

Microoperations are elementary operations performed on data stored in registers.

  • Register Transfer Microoperations: Data transfer between registers.
  • Arithmetic Microoperations: Perform arithmetic operations like addition, subtraction, increment, decrement.
  • Logic Microoperations: Perform bitwise operations like AND, OR, NOT, XOR.
  • Shift Microoperations: Perform shift operations like logical shift, circular shift, arithmetic shift.

Arithmetic Logic Shift Unit (ALSU)

Combines arithmetic, logical, and shifting operations into a single unit for efficient processing.
