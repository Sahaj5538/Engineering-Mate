AC Circuits

Sinusoidal Signals

1. Representation in Time Domain

v(t) = Vm sin(ωt + φ)

  • Vm = Peak value
  • ω = Angular frequency = 2πf (rad/s)
  • f = Frequency in Hz
  • φ = Phase angle
  • t = Time in seconds

2. Representation in Phasor Domain

Phasor: V = Vm∠φ

Phasors simplify AC steady-state analysis by removing time dependence.

Values: Average and RMS

1. Average Value

Vavg = (2Vm) / π

2. RMS Value

Vrms = Vm / √2

Similarly, Irms = Im / √2

Phasors and Sinusoidal Steady-State Analysis

  • Resistor (R): V = IR (same phase)
  • Inductor (L): V = jωLI (current lags voltage by 90°)
  • Capacitor (C): V = I / (jωC) (current leads voltage by 90°)

Impedance (Z):
ZR = R, ZL = jωL, ZC = 1 / (jωC)

Power in AC Circuits

1. Real Power (P)

P = Vrms Irms cosφ

2. Reactive Power (Q)

Q = Vrms Irms sinφ

3. Apparent Power (S)

S = Vrms Irms

Relationship: S = √(P2 + Q2)

4. Power Factor (pf)

Power Factor = cosφ

- Leading pf: Capacitive load
- Lagging pf: Inductive load

AC Circuit Analysis

1. RL Circuit

Impedance: Z = R + jωL

Phase: Current lags voltage.

2. RC Circuit

Impedance: Z = R - j / (ωC)

Phase: Current leads voltage.

3. RLC Circuit

(a) Series RLC

Impedance: Z = R + j(ωL - 1/(ωC))

Resonance: ωL = 1/(ωC)

(b) Parallel RLC

Admittance: Y = 1/Z

Resonance

1. Series Resonance

Occurs when ωL = 1/(ωC)

Resonant frequency: fr = 1 / (2π√LC)

At resonance, Z = R (minimum) and current is maximum.

2. Parallel Resonance

Occurs when reactive admittance is zero.

At resonance, impedance is maximum, and current drawn is minimum.
