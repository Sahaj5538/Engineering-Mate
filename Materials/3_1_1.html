Computer Networks: OSI and TCP/IP Models

Introduction to Computer Networks

A computer network is a system that connects two or more computers so they can share resources and information. Networking allows computers to communicate efficiently and securely over short or long distances.

Types of Networks:

  • LAN (Local Area Network)
  • WAN (Wide Area Network)
  • MAN (Metropolitan Area Network)
  • Wireless Networks (Wi-Fi, Cellular)

OSI Model (Open Systems Interconnection Model)

The OSI Model is a conceptual framework used to understand and implement networking protocols in seven distinct layers. Each layer serves a specific function.

7 Layers of OSI Model:

  • Layer 7 - Application Layer: Provides network services directly to applications (e.g., HTTP, FTP).
  • Layer 6 - Presentation Layer: Translates data formats between applications and the network (e.g., encryption, compression).
  • Layer 5 - Session Layer: Manages sessions or connections between computers (e.g., establishing, maintaining, ending sessions).
  • Layer 4 - Transport Layer: Provides reliable data transfer (e.g., TCP, UDP).
  • Layer 3 - Network Layer: Handles routing and forwarding (e.g., IP addressing, routers).
  • Layer 2 - Data Link Layer: Provides node-to-node data transfer and handles error correction (e.g., MAC address, switches).
  • Layer 1 - Physical Layer: Deals with the physical connection between devices (e.g., cables, switches, voltage levels).

Key Points:

  • Each layer serves the layer above it and is served by the layer below it.
  • Helps standardize network functions for interoperability.

TCP/IP Model

The TCP/IP Model is a simpler, four-layer model used to conceptualize how data should travel across networks. It forms the basis of the internet and most modern networks.

4 Layers of TCP/IP Model:

  • Application Layer: Combines OSI's Application, Presentation, and Session layers. (e.g., HTTP, SMTP, FTP)
  • Transport Layer: Similar to OSI's Transport layer. (e.g., TCP, UDP)
  • Internet Layer: Matches OSI's Network layer. (e.g., IP protocol)
  • Network Access Layer: Combines OSI's Data Link and Physical layers. (e.g., Ethernet, Wi-Fi)

Key Points:

  • More practical and widely implemented than OSI.
  • Focuses on protocols used on the internet.
  • Provides end-to-end communications specifying how data should be packetized, addressed, transmitted, routed, and received.

Comparison: OSI vs TCP/IP

Aspect OSI Model TCP/IP Model
Layers 7 Layers 4 Layers
Development Theoretical model Practical, based on standard protocols
Usage Rarely implemented Widely used in real-world networks
Approach Prescriptive (how networks should work) Descriptive (how networks actually work)
