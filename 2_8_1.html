HTML Common Tags

1. Lists

HTML provides three types of lists:

  • Ordered List (<ol>): A list with numbers.
  • Unordered List (<ul>): A list with bullets.
  • Definition List (<dl>): A list of terms and descriptions.

2. Tables

Tables are used to display data in rows and columns using the <table> tag. Important tags include:

  • <tr> - Table row
  • <td> - Table data cell
  • <th> - Table header cell
  • <thead>, <tbody>, <tfoot> - Table sections

3. Images

Images are added using the <img> tag. Important attributes:

  • src - Image URL
  • alt - Alternative text
  • width and height - Image dimensions

4. Forms

Forms collect user input. Key elements:

  • <form> - Defines the form
  • <input> - Input field
  • <textarea> - Multi-line text input
  • <select> and <option> - Dropdowns
  • <button> - Submit or reset actions

5. Frames

Frames divide the browser window into multiple sections. Now outdated and replaced by CSS/JS techniques, but classic tags:

  • <frameset> - Defines a set of frames
  • <frame> - Defines a single frame
  • <iframe> - Inline frame, still widely used

Cascading Style Sheets (CSS)

CSS is used to control the layout and appearance of web pages. Key concepts:

  • Selectors: Target HTML elements (e.g., p, #id, .class).
  • Properties: Define the styles (e.g., color, font-size, margin).
  • Cascading: Multiple rules can apply to one element; priority is determined by specificity.
  • Box Model: Content + Padding + Border + Margin
  • Media Queries: Make designs responsive for different screen sizes.

Introduction to JavaScript

JavaScript is a scripting language used to create dynamic and interactive web pages. It runs in the browser and can manipulate HTML and CSS.

Objects and Functions in JavaScript

Objects: Collections of properties and methods. Example:

let car = {
    brand: "Toyota",
    model: "Camry",
    start: function() {
        console.log("Car started");
    }
};

Functions: Blocks of code designed to perform tasks. Example:

function greet(name) {
    alert("Hello, " + name + "!");
}

Manipulating DOM (Document Object Model)

DOM is a tree structure representing HTML elements. JavaScript can change, add, or remove elements:

  • document.getElementById()
  • document.getElementsByClassName()
  • document.querySelector()
  • element.innerHTML - Change content
  • element.style - Change CSS styles

HTML DOM Events

Events allow interaction. Common events include:

  • onclick - User clicks an element
  • onmouseover - Mouse over an element
  • onkeydown - Key press
  • onsubmit - Form submission

Example:

document.getElementById("myButton").onclick = function() {
    alert("Button clicked!");
};

Learning Ajax

AJAX (Asynchronous JavaScript and XML) allows web pages to update asynchronously by exchanging data with a web server in the background.

Basic Ajax Example:

let xhr = new XMLHttpRequest();
xhr.open("GET", "data.json", true);
xhr.onload = function() {
    if (xhr.status == 200) {
        console.log(xhr.responseText);
    }
};
xhr.send();

jQuery Programming

Selectors & Mouse Events

jQuery simplifies JavaScript coding:

$(document).ready(function(){
    $("p").click(function(){
        $(this).hide();
    });
});

Form Events

Form events include:

  • submit()
  • focus()
  • blur()
$("#form").submit(function(){
    alert("Form submitted!");
});

DOM Manipulation

  • text() - Get or set text
  • html() - Get or set HTML content
  • attr() - Get or set attributes
$("#demo").text("Hello jQuery!");

Effects & Animation

jQuery can create effects:

  • hide(), show()
  • fadeIn(), fadeOut()
  • slideUp(), slideDown()

Traversing & Filtering

Find and filter elements easily:

  • find() - Find elements inside others
  • children() - Get child elements
  • filter() - Filter based on criteria
$("#list").children(".active").css("color", "red");
