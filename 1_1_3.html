Matrix Decompositions and Linear Algebra

LU Decomposition

LU decomposition decomposes a matrix A into the product of a lower triangular matrix L and an upper triangular matrix U, i.e., A = LU. This is particularly useful for solving linear systems Ax = b efficiently.

Steps:

  • First, solve Ly = b for y using forward substitution.
  • Then, solve Ux = y for x using backward substitution.

Cholesky Decomposition

Cholesky decomposition is a special case of LU decomposition applicable to symmetric positive definite matrices. It decomposes A into the product of a lower triangular matrix L and its transpose LT, i.e., A = LLT.

Cholesky decomposition is more efficient and numerically stable than LU decomposition for such matrices.

Gram-Schmidt Orthonormalization Process

The Gram-Schmidt process is an algorithm for orthonormalizing a set of vectors in an inner product space.

Given a set of linearly independent vectors {v₁, v₂, ..., vₙ}, it constructs an orthonormal set {u₁, u₂, ..., uₙ}:

  • u₁ = v₁ / |v₁|
  • For i = 2, ..., n:
    • wᵢ = vᵢ - Σ (⟨vᵢ, uⱼ⟩ uⱼ) for j = 1 to i-1
    • uᵢ = wᵢ / |wᵢ|

This is fundamental for QR factorization and eigenvalue computations.

QR Factorization

QR factorization decomposes a matrix A into the product of an orthogonal matrix Q and an upper triangular matrix R, i.e., A = QR.

Applications:

  • Solving linear least squares problems.
  • Eigenvalue problems.

One method to compute QR factorization is applying the Gram-Schmidt process to the columns of A.

Eigen Decomposition of Symmetric Matrices

For a symmetric matrix A, eigen decomposition is given by:

A = PDP-1

Where D is a diagonal matrix containing eigenvalues and P is a matrix whose columns are eigenvectors.

For symmetric matrices, P can be orthogonal (Q), so:

A = QDQT

This simplifies computations like matrix powers and solving linear systems.

Singular Value Decomposition (SVD)

The Singular Value Decomposition of any matrix A (square or not) is:

A = UΣVT

  • U: orthogonal matrix (left singular vectors).
  • Σ: diagonal matrix (singular values).
  • V: orthogonal matrix (right singular vectors).

Applications include dimensionality reduction, image compression, and solving least squares problems.

Least Squares Solution (QR Factorization and Generalized Inverse)

The least squares problem seeks to minimize |Ax - b|².

Methods:

  • Using QR Factorization: If A = QR, solve Rx = QTb using backward substitution.
  • Using the Generalized Inverse: The solution is x = A+b, where A+ = VΣ+UT and Σ+ is the pseudo-inverse of Σ.
